HTML, CSS, and Bootstrap

HTML

Hypertext Markup Language

HTML is a markup language. It is not a programming language. It is the base of format. It is used to build the structure of a Web page.

CSS

Cascading Style Sheets

CSS is not a markup language. It is a stylesheet language. It adds color to an HTML document. It is used to design the appearance in a more aesthetic way.

Bootstrap

Bootstrap Version 4

Bootstrap is a framework. It is not a programming language. It is free and opensource. It can be used both online and offline.
